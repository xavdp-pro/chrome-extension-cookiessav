# Cookie Saver Extension

Extension Chrome pour sauvegarder les cookies d'un site en format JSON avec sélection manuelle.

## Fonctionnalités

- Affichage de tous les cookies du site actuel
- Sélection manuelle via une modale (cocher/décocher)
- Recherche de cookies par nom, valeur ou domaine
- Export en JSON (auth, prefs - sans analytics)

## Installation

1. `chrome://extensions/` → Mode développeur
2. Charger l'extension non empaquetée → sélectionner ce dossier

## Utilisation

1. Naviguer sur le site
2. Cliquer sur l'icône → Sélectionner les cookies
3. Cocher/décocher, puis Sauvegarder
